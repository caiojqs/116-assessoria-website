# 116 Assessoria - Website Institucional

Website institucional da **116 Assessoria**, empresa especializada em assessoria de importação.

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 (Custom Properties, Grid, Flexbox, animações)
- JavaScript Vanilla (ES6+)
- Google Fonts (Montserrat + Raleway)
- Font Awesome 6

## 📁 Estrutura

```
116-assessoria/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos completos
├── js/
│   └── main.js         # Interatividade e animações
├── img/                # Diretório para imagens
├── package.json        # Para servidor local
└── README.md           # Este arquivo
```

## 🎨 Design

- **Cores:** Azul (#1a3a6b), Branco (#ffffff), Vermelho (#c0392b)
- **Fontes:** Montserrat (títulos) e Raleway (corpo)
- **Layout:** Responsivo (mobile-first adaptado)

## 📋 Seções

1. **Home** - Hero com CTA
2. **Serviços** - 6 cards de serviços
3. **Parceiros** - Estatísticas e regiões
4. **Sobre Nós** - Missão, valores
5. **Eventos** - Próximos eventos
6. **Contato** - Formulário + informações

## 🛠️ Como usar

```bash
# Com Node.js instalado:
npm install
npm start

# Ou simplesmente abra index.html no navegador
```

## 📱 Responsividade

O site é totalmente responsivo e otimizado para:
- Desktop (1200px+)
- Tablet (768px - 992px)
- Mobile (até 768px)
